Org Reveal

Org Reveal allows you to export Orgmode documents to presentations. Org Reveal has the following GIT repository and documentation.

My configuration

Here is my personal configuration back to when I used this; Place this at the top of your OrgMode document then export with org-export-dispach 

:reveal_properties:
#+reveal_root: https://cdn.jsdelivr.net/npm/reveal.js
#+reveal_reveal_js_version: 4
#+reveal_theme: serif
#+options: timestamp:nil toc:1 num:nil
:end:

Every line per space

You can add this functionality with the following line: More information could be found here: https://revealjs.com/fragments/ 

#+aatr_reveal: :frag (appear)

images

You can add and do basic manipulation with images with thefollowing line. 

#+aatr_html: :width 45% :align center
