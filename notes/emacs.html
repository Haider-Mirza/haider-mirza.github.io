IDE

I Love Programming, and as a result, you would want a great IDE. What's a better IDE than Emacs? LSP-mode allows you to use Language Server Protocol (LSP) in Emacs. (That's pretty insane) I mainly use LSP-mode with clangd (you can see more about clangd in CMake)

Some useful keybinds

Note that on my system, 'CTRL' is re-keybinded to 'CAPS LOCK' for erganomic and speed benifits

C-j for return C-i for tab

Email

Issues

Emacs can't find guix package

If I ever encounter that Emacs cant find a package installed by Guix, I found a tempoary solution online: 

(let ((default-directory  "/home/haider/.guix-extra-profiles/mail/mail/share/emacs/site-lisp"))
  (normal-top-level-add-subdirs-to-load-path))
