Org-roam

About

One of the best ways to think and work through problems is to have a written dialogue with yourself. Using Org-roam you have a better way to log your thoughts from the day, link them to existing notes, and easily review the notes for any day, all with Org Roam. I use Org-roam as a place I can place information in my eternal quest for knowledge. Org-roam is built over org-mode (That's partially what makes it so awesome).

Documentation

You can get documentation from these links: Main Documentation Get Started with Org-roam - Systemcrafters Capturing notes efficiently with Org Roam - Systemcrafters

NOTE:

To install, make sure you have gcc AND gcc-toolchain (If you use Guix) installed.

Important information

  • In my configuration my Org roam notes go to the ~/Notes/ directory.

Inserting node links using completions

You can also use M-x completion-at-point to complete the names of nodes inside of double-square brackets [[]].

Keybinds

Keybind Function
SPC - orf find a Org-roam note
SPC - orc creates a Org-roam note
SPC - ori insert a link to a Org-roam note
