How to build the game

If you use the GNU Guix package manager, you can run the command “guix shell -m manifest.scm” to create a seperate programming environment. You can read more about this feature in the Guix Manual: Guix Shell

If you dont use the Guix Package manager and cannot use the pre-built binary located in the ./bin/. Launch a terminal and go into the project’s root directory then run the command, make all. Make sure you have all of the dependencies required installed. These are listed in the manifest.scm file.
