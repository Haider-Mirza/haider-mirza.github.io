Spinter

OpenGL Manipulation

How to Hack It

If you're interested in cloning the code to try it out, I recommend installing the GNU Guix package manager. Using Guix will allow you to install all compilation dependencies like so: 

cd ~/path/to/Spinter
guix shell -m manifest.scm

You will now be in a shell with all of the dependencies configured. You can now build and run the code: 

chmod +x ./start.sh
./start.sh
# This is only needed the first time you clone the project
./build-run.sh
# This is to build and run the project

Editor Setup

I use GNU Emacs to hack on and interface with this project. Here's what you need to use for a similar setup:

  • Emacs 28 (27 is sufficient)
  • lsp-mode with the ccls or (the preferred and the one I use) clam language server for C/C++ development
