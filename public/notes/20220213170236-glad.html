GLAD

GLAD is extention loader Library to load OpenGL and do things.

Installation

You actually have to generate the GLAD files using the following website: glad.david.de

Get Working

Make sure you use Both this include and define. 

#define GLFW_INCLUDE_NONE
#include <glad/gl.h>

NOTE

Be sure to include GLAD before GLFW. The include file for GLAD includes the required OpenGL headers behind the scenes (like GL/gl.h) so be sure to include GLAD before other header files that require OpenGL (like GLFW).
